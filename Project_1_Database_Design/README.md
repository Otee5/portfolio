# Project 1 — Relational Database Design (Oracle SQL)

**Tools:** Oracle SQL 19c, SQL scripts  

## Objective
Design a normalized relational database to manage data efficiently.

## What I built
- Normalized relational schema with primary & foreign keys
- SQL scripts for table creation and example queries
- Query examples demonstrating joins and aggregate functions

## Files
- `sql_queries.sql` — CREATE TABLE and sample SELECT queries
- `screenshots/` — results and ER diagrams

## How to run
1. Open Oracle SQL or a compatible SQL client.
2. Run `sql_queries.sql` to create tables and sample data.
3. Execute sample queries included in the file.
